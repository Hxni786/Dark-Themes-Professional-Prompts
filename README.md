## ⚡ Key Features

- **Advanced Design Systems**: Complete token systems (colors, typography, spacing, shadows).
- **Component DNA**: Detailed specifications for buttons, cards, inputs, and navs.
- **Micro-Interactions**: Specific guidelines for hover states, transitions, and animations.
- **LLM-Ready**: Structured with `<role>` and `<design-system>` tags for perfect interpretation by AI models.
- **Visual Previews**: Each prompt is paired with a high-fidelity image reflecting the style's potential.

---

## 🚀 How to Use These Prompts

These prompts are designed to be used with Advanced Agentic AI systems (like Gemini, Claude, or GPT-4) to guide the creation of complex UI.

1. **Select a Style**: Choose a `.md` file from the repository.
2. **Copy the Content**: Copy the entire text of the file.
3. **Paste into LLM**: Paste it as system context or a starting prompt for your AI developer.
4. **Build Your UI**: Start issuing build commands (e.g., "Create a pricing section in the Terminal CLI style").

## 🛠️ Recommended Tech Stack

While these styles are platform-agnostic, they are optimized for modern frontend development:

- **Styling**: Tailwind CSS (extensively referenced in tokens)
- **Animations**: Framer Motion, GSAP, or CSS Keyframes
- **Icons**: Lucide React, FontAwesome, or custom SVGs
